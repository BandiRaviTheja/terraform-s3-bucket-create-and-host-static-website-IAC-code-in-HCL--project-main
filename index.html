🚀 DevOps Cloud Deployment Project

Static Website Hosted on AWS S3 using Terraform and Jenkins CI/CD Pipeline

About This Project

This project demonstrates how Infrastructure as Code (IaC) and CI/CD automation can be used to deploy a static website on AWS. Terraform provisions the infrastructure while Jenkins automates the deployment process.

☁ AWS S3

Secure and scalable cloud storage used for hosting the static website.

⚙ Terraform

Infrastructure as Code tool used to provision and manage AWS resources.

🚀 Jenkins

CI/CD automation server used for continuous deployment.
